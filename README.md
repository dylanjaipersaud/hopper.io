## Development

To get started, make sure you have Node v19.9.0 and NPM v9.6.3 installed. Then,

```bash
$ npm install
$ npm run build-dev
```

on your local machine.


## Tests

To run the tests for this this project, simply

```bash
$ npm install
$ npm test
```
